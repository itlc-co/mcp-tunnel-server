# mcp-tunnel-server
 MCP（Model Context Protocol）内网穿透服务集成到 Claude Code 开发环境中
# 核心架构
  Claude Code 通过 MCP 服务器 与外部世界交互，MCP 服务器可以内嵌隧道客户端，从而打通内外网：

  ┌─────────────────┐     MCP stdio      ┌──────────────────┐
  │  Claude Code     │◄──────────────────►│  MCP Server      │
  │  (你的开发机)     │                    │  (内置 tunnel)    │
  └─────────────────┘                    └──────┬───────────┘
                                                │
                            ┌───────────────────┼───────────┐
                            │  隧道客户端        │           │
                            │  (frpc/ngrok)     │           │
                            └───────┬───────────┘           │
                                    │                      │
                            ════════╪══════════  (公网/内网)
                                    │
                            ┌───────┴───────────┐
                            │  隧道服务端/入口    │
                            │  (frps/Cloudflare) │
                            └───────┬───────────┘
                                    │
                            ┌───────┴───────────┐
                            │  内部服务           │
                            │  (内网API/DB/...)  │
                            └───────────────────┘

  两条路径：
  - 内→外（暴露）：把本地服务通过隧道暴露到公网
  - 外→内（穿透）：通过隧道访问内网中未直接暴露的服务
